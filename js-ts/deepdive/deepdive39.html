                                                                                                                          # 1.1. HTML мҡ”мҶҢмҷҖ л…ёл“ң к°қмІҙ

                                                                                                                          HTML мҡ”мҶҢлҠ” HTML л¬ём„ңлҘј кө¬м„ұн•ҳлҠ” к°ңлі„м Ғмқё мҡ”мҶҢлҘј мқҳлҜё

                                                                                                                          HTMLмҡ”мҶҢмқҳ м–ҙнҠёлҰ¬л·°нҠёлҠ” м–ҙнҠёлҰ¬л·°нҠё л…ёл“ңлЎң, HTML мҡ”мҶҢмқҳ н…ҚмҠӨнҠё мҪҳн…җмё лҠ” н…ҚмҠӨнҠё л…ёл“ңлЎң ліҖнҷҳ

                                                                                                                          HTML л¬ём„ңлҠ” HTML мҡ”мҶҢл“Өмқҳ м§‘н•©мңјлЎң мқҙлӨ„м§Җл©° мӨ‘мІ©кҙҖкі„лҘј к°–лҠ”лӢӨ
                                                                                                                          мқҙ л•Ң мҡ”мҶҢ к°„м—җлҠ” мӨ‘мІ© кҙҖкі„м—җ мқҳн•ҙ кі„мёөм Ғ л¶Җмһҗ(parent-child) кҙҖкі„к°Җ нҳ•м„ұлҗңлӢӨ.

                                                                                                                          # нҠёлҰ¬ мһҗлЈҢкө¬мЎ°

                                                                                                                          нҠёлҰ¬ мһҗлЈҢкө¬мЎ°лҠ” л¶ҖлӘЁ л…ёл“ңмҷҖ мһҗмӢқ л…ёл“ңлЎң кө¬м„ұлҗҳм–ҙ л…ёл“ң к°„мқҳ кі„мёөм Ғ кө¬мЎ°лҘј н‘ңнҳ„н•ҳлҠ” л№„м„ нҳ• мһҗлЈҢкө¬мЎ°лҘј л§җн•ңлӢӨ.

                                                                                                                          н•ҳлӮҳмқҳ мөңмғҒмң„ л…ёл“ңм—җм„ң мӢңмһ‘н•ңлӢӨ. мөңмғҒмң„ л…ёл“ңлҘј лЈЁнҠёл…ёл“ңлқј н•ңлӢӨ.
                                                                                                                          мһҗмӢқ л…ёл“ңк°Җ м—ҶлҠ” л…ёл“ңлҘј лҰ¬н”„ л…ёл“ңлқј н•ңлӢӨ.

                                                                                                                          л…ёл“ң к°қмІҙл“ӨлЎң кө¬м„ұлҗң нҠёлҰ¬ мһҗлЈҢкө¬мЎ°лҘј DOM нҠёлҰ¬лқјкі  л¶ҖлҘҙкё°лҸ„ н•ңлӢӨ.

                                                                                                                          # 1.3. л…ёл“ң к°қмІҙмқҳ мғҒмҶҚ кө¬мЎ°

                                                                                                                          DOMмқ„ кө¬м„ұн•ҳлҠ” л…ёл“ң к°қмІҙмқҳ мғҒмҶҚ кө¬мЎ°лҠ” лӢӨмқҢкіј к°ҷлӢӨ.

                                                                                                                          input мҡ”мҶҢ л…ёл“ңмқҳ к°қмІҙлҠ” н”„лЎңнҶ нғҖмһ… мІҙмқём—җ мһҲлҠ” лӘЁл“  н”„лЎңнҶ нғҖмһ…мқҳ н”„лЎңнҚјнӢ°лӮҳ л©”мҠӨл“ңлҘј мғҒмҶҚл°ӣм•„ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

                                                                                                                          <!DOCTYPE html>
<html>
  <body>
    <input type="text" />
    <script>
      // input мҡ”мҶҢ л…ёл“ң к°қмІҙлҘј м„ нғқ
      const $input = document.querySelector("input");

      // input мҡ”мҶҢ л…ёл“ң к°қмІҙмқҳ н”„лЎңнҶ нғҖмһ… мІҙмқё
      console.log(
        Object.getPrototypeOf($input) === HTMLInputElement.prototype,
        Object.getPrototypeOf(HTMLInputElement.prototype) ===
          HTMLElement.prototype,
        Object.getPrototypeOf(HTMLElement.prototype) === Element.prototype,
        Object.getPrototypeOf(Element.prototype) === Node.prototype,
        Object.getPrototypeOf(Node.prototype) === EventTarget.prototype,
        Object.getPrototypeOf(EventTarget.prototype) === Object.prototype
      ); // лӘЁл‘җ true
    </script>
  </body>
</html>

                                                                                                                          мқҙлҹ¬н•ң нҠ№м„ұмқ„ лӮҳнғҖлӮҙлҠ” кё°лҠҘл“Өмқ„ мғҒмҶҚмқ„ нҶөн•ҙ м ңкіөл°ӣлҠ”лӢӨ.

                                                                                                                          мӣ№ л¬ём„ңмқҳ мҡ”мҶҢ мӨ‘м—җм„ң HTML мҡ”мҶҢлҘј н‘ңнҳ„н•ҳлҠ” к°қмІҙHTMLElement
                                                                                                                          HTML мҡ”мҶҢ мӨ‘м—җм„ң input мҡ”мҶҢлҘј н‘ңнҳ„н•ҳлҠ” к°қмІҙHTMLInputElement

                                                                                                                          мҰү л…ёл“ң нғҖмһ…м—җ л”°лқј н•„мҡ”н•ң кё°лҠҘмқ„ н”„лЎңнҚјнӢ°мҷҖ л©”м„ңл“ңмқҳ м§‘н•©мқё DOM APIлЎң м ңкіөн•ңлӢӨ.

                                                                                                                          мқҙ DOM APIлҘј нҶөн•ҙ HTMLмқҳ кө¬мЎ°лӮҳ лӮҙмҡ© лҳҗлҠ” мҠӨнғҖмқј л“ұмқ„ лҸҷм ҒмңјлЎң мЎ°мһ‘н•  мҲҳ мһҲлӢӨ.

                                                                                                                          # 2.6. HTMLCollectionкіј NodeList

                                                                                                                          HTMLCollectionкіј NodeListмқҳ мӨ‘мҡ”н•ң нҠ№м§•мқҖ л…ёл“ң к°қмІҙмқҳ мғҒнғң ліҖнҷ”лҘј мӢӨмӢңк°„мңјлЎң л°ҳмҳҒн•ҳлҠ” мӮҙм•„ мһҲлҠ” к°қмІҙлқјлҠ” кІғмқҙлӢӨ.

                                                                                                                          # HTMLCollection

                                                                                                                          <!DOCTYPE html>
<head>
  <style>
    .red {
      color: red;
    }
    .blue {
      color: blue;
    }
  </style>
</head>
<html>
  <body>
    <ul id="fruits">
      <li class="red">Apple</li>
      <li class="red">Banana</li>
      <li class="red">Orange</li>
    </ul>
    <script>
      // class к°’мқҙ 'red'мқё мҡ”мҶҢ л…ёл“ңлҘј лӘЁл‘җ нғҗмғүн•ҳм—¬ HTMLCollection к°қмІҙм—җ лӢҙм•„ л°ҳнҷҳн•ңлӢӨ.
      const $elems = document.getElementsByClassName("red");
      // мқҙ мӢңм җм—җ HTMLCollection к°қмІҙм—җлҠ” 3к°ңмқҳ мҡ”мҶҢ л…ёл“ңк°Җ лӢҙкІЁ мһҲлӢӨ.
      console.log($elems); // HTMLCollection(3) [li.red, li.red, li.red]

      // HTMLCollection к°қмІҙмқҳ лӘЁл“  мҡ”мҶҢмқҳ class к°’мқ„ 'blue'лЎң ліҖкІҪн•ңлӢӨ.
      for (let i = 0; i < $elems.length; i++) {
        $elems[i].className = "blue";
      }

      // HTMLCollection к°қмІҙмқҳ мҡ”мҶҢк°Җ 3к°ңм—җм„ң 1к°ңлЎң ліҖкІҪлҗҳм—ҲлӢӨ.
      console.log($elems); // HTMLCollection(1) [li.red]
    </script>
  </body>
</html>

                                                                                                                          мң„ мҳҲм ңлҠ” HTMLCollection к°қмІҙлҘј for л¬ёмңјлЎң мҲңнҡҢн•ҳл©° className н”„лЎңнҚјнӢ°лҘј мӮ¬мҡ©н•ҳм—¬
                                                                                                                          лӘЁл“  мҡ”мҶҢмқҳ class к°’мқ„ 'red'м—җм„ң 'blue'лЎң ліҖкІҪн•ңлӢӨ.

                                                                                                                          $elems.lengthлҠ” 3мқҙлҜҖлЎң for л¬ёмқҳ мҪ”л“ң лё”лЎқмқҖ 3лІҲ л°ҳліөлҗңлӢӨ.

                                                                                                                          HTMLCollection к°қмІҙлҠ” мӢӨмӢңк°„мңјлЎң л…ёл“ң к°қмІҙмқҳ мғҒнғңлҘј ліҖкІҪмқ„ л°ҳмҳҒн•ҳм—¬ мҡ”мҶҢлҘј м ңкұ°н•  мҲҳ мһҲкё° л•Ңл¬ём—җ
                                                                                                                          for л¬ёмқ„ мҲңнҡҢн•ҳл©ҙм„ң л…ёл“ң к°қмІҙмқҳ мғҒнғңлҘј ліҖкІҪн•ҙм•јн•  л•Ң мЈјмқҳн•ҙм•јн•ңлӢӨ.
                                                                                                                          мқҙ л¬ём ңлҠ” forл¬ёмқ„ м—ӯл°©н–ҘмңјлЎң мҲңнҡҢн•ҳлҠ” л°©лІ•мңјлЎң нҡҢн”јн•  мҲҳ мһҲлӢӨ.

                                                                                                                          // for л¬ёмқ„ м—ӯл°©н–ҘмңјлЎң мҲңнҡҢ
for (let i = $elems.length - 1; i >= 0; i--) {
  $elems[i].className = "blue";
}

                                                                                                                          while л¬ёмқ„ мӮ¬мҡ©н•ҳм—¬ HTMLCollection к°қмІҙм—җ л…ёл“ң к°қмІҙк°Җ лӮЁм•„ мһҲм§Җ м•Ҡмқ„ л•Ң к№Ңм§Җ л¬ҙн•ң л°ҳліөн•ҳлҠ” л°©лІ•мңјлЎң нҡҢн”јн•  мҲҳ мһҲлӢӨ.

                                                                                                                          // while л¬ёмңјлЎң HTMLCollectionм—җ мҡ”мҶҢк°Җ лӮЁм•„ мһҲм§Җ м•Ҡмқ„ л•Ңк№Ңм§Җ л¬ҙн•ң л°ҳліө
let i = 0;
while ($elems.length > i) {
  $elems[i].className = "blue";
}

                                                                                                                          мң мӮ¬ л°°м—ҙ к°қмІҙмқҙл©ҙм„ң мқҙн„°лҹ¬лё”мқё HTMLCollection к°қмІҙлҘј л°°м—ҙлЎң ліҖнҷҳн•ҳл©ҙ л¶Җмһ‘мҡ©мқ„ л°ңмғқмӢңнӮӨлҠ”
                                                                                                                          HTMLCollection к°қмІҙлҘј мӮ¬мҡ©н•  н•„мҡ”к°Җ м—Ҷкі  мң мҡ©н•ң л°°м—ҙмқҳ кі м°Ён•ЁмҲҳлҘј мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

                                                                                                                          // мң мӮ¬ л°°м—ҙ к°қмІҙмқҙл©ҙм„ң мқҙн„°лҹ¬лё”мқё HTMLCollectionмқ„ л°°м—ҙлЎң ліҖнҷҳн•ҳм—¬ мҲңнҡҢ
[...$elems].forEach((elem) => (elem.className = "blue"));

                                                                                                                          # NodeList

                                                                                                                          getElementsByTagName, getElementsByClassName л©”м„ңл“ң лҢҖмӢ 
                                                                                                                          querySelectorAll л©”м„ңл“ңлҘј мӮ¬мҡ©н•ҳлҠ” л°©лІ•лҸ„ мһҲлӢӨ.

                                                                                                                          querySelectorAll л©”м„ңл“ңлҠ” DOM м»¬л үм…ҳ к°қмІҙмқё NodeList к°қмІҙлҘј л°ҳнҷҳн•ңлӢӨ.

                                                                                                                          мқҙл•Ң NodeList к°қмІҙлҠ” мӢӨмӢңк°„ л…ёл“ң к°қмІҙмқҳ мғҒнғң ліҖкІҪмқ„ л°ҳмҳҒн•ҳм§Җ м•ҠлҠ” к°қмІҙлӢӨ.

                                                                                                                          // querySelectorAllмқҖ DOM м»¬л үм…ҳ к°қмІҙмқё NodeListлҘј л°ҳнҷҳн•ңлӢӨ.
const $elems = document.querySelectorAll(".red");

// NodeList к°қмІҙлҠ” NodeList.prototype.forEach л©”м„ңл“ңлҘј мғҒмҶҚл°ӣм•„ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.
$elems.forEach((elem) => (elem.className = "blue"));

  </script>
</html>

                                                                                                                          лӢЁ, innerHTML н”„лЎңнҚјнӢ°мҷҖ л§Ҳм°¬к°Җм§ҖлЎң HTML л§ҲнҒ¬м—… л¬ёмһҗм—ҙмқ„ нҢҢмӢұн•ҳлҜҖлЎң нҒ¬лЎңмҠӨ мӮ¬мқҙнҠё мҠӨнҒ¬лҰҪнҢ… кіөкІ©м—җ м·Ём•Ҫн•ҳлӢӨ.

                                                                                                                          # 6.3. л…ёл“ң мғқм„ұкіј м¶”к°Җ

                                                                                                                          DOMмқҖ л…ёл“ңлҘј м§Ғм ‘ мғқм„ұ/мӮҪмһ…/мӮӯм ң/м№ҳнҷҳн•ҳлҠ” л©”м„ңл“ңлҸ„ м ңкіөн•ңлӢӨ.

                                                                                                                          <!DOCTYPE html>
<html>
  <body>
    <ul id="fruits">
      <li>Apple</li>
    </ul>
  </body>
  <script>
    const $fruits = document.getElementById("fruits");

    // 1. мҡ”мҶҢ л…ёл“ң мғқм„ұ
    const $li = document.createElement("li");

    // 2. н…ҚмҠӨнҠё л…ёл“ң мғқм„ұ
    const textNode = document.createTextNode("Banana");

    // 3. н…ҚмҠӨнҠё л…ёл“ңлҘј $li мҡ”мҶҢ л…ёл“ңмқҳ мһҗмӢқ л…ёл“ңлЎң м¶”к°Җ
    $li.appendChild(textNode);

    // 4. $li мҡ”мҶҢ л…ёл“ңлҘј #fruits мҡ”мҶҢ л…ёл“ңмқҳ л§Ҳм§Җл§ү мһҗмӢқ л…ёл“ңлЎң м¶”к°Җ
    $fruits.appendChild($li);
  </script>
</html>

                                                                                                                          # 7. м–ҙнҠёлҰ¬л·°нҠё

                                                                                                                          # 7.1. м–ҙнҠёлҰ¬л·°нҠё л…ёл“ңмҷҖ attributes н”„лЎңнҚјнӢ°

                                                                                                                          HTML мҡ”мҶҢмқҳ лҸҷмһ‘мқ„ м ңм–ҙн•ҳкё° мң„н•ң м¶”к°Җм Ғмқё м •ліҙлҘј м ңкіөн•ҳлҠ” HTML м–ҙнҠёлҰ¬л·°нҠёлҠ” HTML мҡ”мҶҢмқҳ мӢңмһ‘ нғңк·ём—җ
                                                                                                                          м–ҙнҠёлҰ¬л·°нҠё мқҙлҰ„ = "м–ҙнҠёлҰ¬л·°нҠё к°’" нҳ•мӢқмңјлЎң м •мқҳн•ңлӢӨ.

                                                                                                                          <input id="user" type="text" value="ungmo2" />

                                                                                                                          attributes н”„лЎңнҚјнӢ°лҠ” getterл§Ң мЎҙмһ¬н•ҳлҠ” мқҪкё° м „мҡ© м ‘к·јмһҗ н”„лЎңнҚјнӢ°мқҙл©°,
                                                                                                                          мҡ”мҶҢ л…ёл“ңмқҳ лӘЁл“  м–ҙнҠёлҰ¬л·°нҠё л…ёл“ңмқҳ м°ёмЎ°к°Җ лӢҙкёҙ NamedNodeMap к°қмІҙлҘј л°ҳнҷҳн•ңлӢӨ.

                                                                                                                          <!DOCTYPE html>
<html>
  <body>
    <input id="user" type="text" value="ungmo2" />
    <script>
      // мҡ”мҶҢ л…ёл“ңмқҳ attribute н”„лЎңнҚјнӢ°лҠ” мҡ”мҶҢ л…ёл“ңмқҳ лӘЁл“  м–ҙнҠёлҰ¬л·°нҠё л…ёл“ңмқҳ м°ёмЎ°к°Җ лӢҙкёҙ NamedNodeMap к°қмІҙлҘј л°ҳнҷҳн•ңлӢӨ.
      const { attributes } = document.getElementById("user");
      console.log(attributes);
      // NamedNodeMap 
      //{0: id, 1: type, 2: value, id: id, type: type, value: value, length: 3}

      // м–ҙнҠёлҰ¬л·°нҠё к°’ м·Ёл“қ
      console.log(attributes.id.value); // user
      console.log(attributes.type.value); // text
      console.log(attributes.value.value); // ungmo2
    </script>
  </body>
</html>

                                                                                                                          # 7.2. HTML м–ҙнҠёлҰ¬л·°нҠё мЎ°мһ‘

                                                                                                                          attributes н”„лЎңнҚјнӢ°лҠ” getterл§Ң мЎҙмһ¬н•ҳлҠ” мқҪкё° м „мҡ© м ‘к·јмһҗ н”„лЎңнҚјнӢ°лЎң HTML м–ҙнҠёлҰ¬л·°нҠё к°’мқ„ м·Ёл“қн•  мҲҳ мһҲм§Җл§Ң ліҖкІҪн•  мҲҳлҠ” м—ҶлӢӨ.

                                                                                                                          <!DOCTYPE html>
<html>
  <body>
    <input id="user" type="text" value="ungmo2" />
    <script>
      const $input = document.getElementById("user");

      // value м–ҙнҠёлҰ¬л·°нҠё к°’мқ„ м·Ёл“қ
      const inputValue = $input.getAttribute("value");
      console.log(inputValue); // ungmo2

      // value м–ҙнҠёлҰ¬л·°нҠё к°’мқ„ ліҖкІҪ
      $input.setAttribute("value", "foo");
      console.log($input.getAttribute("value")); // foo
    </script>
  </body>
</html>

                                                                                                                          # 8. мҠӨнғҖмқј

                                                                                                                          # 8.1. мқёлқјмқё мҠӨнғҖмқј мЎ°мһ‘

                                                                                                                          HTMLElement.prototype.style н”„лЎңнҚјнӢ°лҠ” setterмҷҖ getter лӘЁл‘җ мЎҙмһ¬н•ҳлҠ” м ‘к·јмһҗ н”„лЎңнҚјнӢ°лЎңм„ң
                                                                                                                          мҡ”мҶҢ л…ёл“ңмқҳ мқёлқјмқё мҠӨнғҖмқјмқ„ м·Ёл“қн•ҳкұ°лӮҳ м¶”к°Җ лҳҗлҠ” ліҖкІҪн•ңлӢӨ.

                                                                                                                          style н”„лЎңнҚјнӢ°лҘј м°ёмЎ°н•ҳл©ҙ CSSStyleDeclaration к°қмІҙлҠ” лӢӨм–‘н•ң CSS н”„лЎңнҚјнӢ°м—җ лҢҖмқ‘н•ҳлҠ” н”„лЎңнҚјнӢ°лҘј к°Җм§Җкі  мһҲмңјл©°,
                                                                                                                          мқҙ н”„лЎңнҚјнӢ°м—җ к°’мқ„ н• лӢ№н•ҳл©ҙ н•ҙлӢ№ CSS н”„лЎңнҚјнӢ°к°Җ мқёлқјмқё мҠӨнғҖмқјлЎң HTML мҡ”мҶҢм—җ м¶”к°Җлҗҳкұ°лӮҳ ліҖкІҪлҗңлӢӨ.

                                                                                                                          $div.style.backgroundColor = "yellow";

                                                                                                                          # 8.2. нҒҙлһҳмҠӨ мЎ°мһ‘

                                                                                                                          HTML мҡ”мҶҢмқҳ class м–ҙнҠёлҰ¬л·°нҠё к°’мқ„ ліҖкІҪн•ҳм—¬ HTML мҡ”мҶҢмқҳ мҠӨнғҖмқјмқ„ ліҖкІҪн•  мҲҳлҸ„ мһҲлӢӨ.
                                                                                                                          мқҙл•Ң HTML мҡ”мҶҢмқҳ class м–ҙнҠёлҰ¬л·°нҠёлҘј мЎ°мһ‘н•ҳл Өл©ҙ class м–ҙнҠёлҰ¬л·°нҠём—җ лҢҖмқ‘н•ҳлҠ” мҡ”мҶҢ л…ёл“ңмқҳ DOM н”„лЎңнҚјнӢ°лҘј мӮ¬мҡ©н•ңлӢӨ.

                                                                                                                          # className

                                                                                                                          <!DOCTYPE html>
<html>
  <head>
    <style>
      .box {
        width: 100px;
        height: 100px;
        background-color: antiquewhite;
      }
      .red {
        color: red;
      }
      .blue {
        color: blue;
      }
    </style>
  </head>
  <body>

      const $box = document.querySelector(".box");

      console.log($box.className); // 'box red'

      // .box мҡ”мҶҢмқҳ class м–ҙнҠёлҰ¬л·°нҠё к°’ мӨ‘м—җм„ң 'red'л§Ң 'blue'лЎң ліҖкІҪ
      $box.className = $box.className.replace("red", "blue");
    </script>
  </body>
</html>

      // classListк°Җ л°ҳнҷҳн•ҳлҠ” DOMTokenList к°қмІҙлҠ” HTMLCollectionкіј NodeListмҷҖ к°ҷмқҙ
